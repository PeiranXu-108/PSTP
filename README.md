# 二维政治光谱测试

通过40道题，帮助你了解自己在**经济**与**社会权威**两个维度上的倾向，并在二维坐标上标出你的位置。

## 你能做什么

- **开始测试**：阅读说明后作答，约 40 题，可分多页完成。
- **查看结果**：在坐标图上看到你的落点，并阅读对应象限的简要说明（价值观倾向、相近意识形态与政党参考）。
- **保存与分享**：可将结果整理为图片下载，便于保存或转发。
- **查看统计**：在本地汇总你多次测试的分布与记录（仅保存在本机浏览器）。

## 使用说明

- 所有答题与结果均在你当前使用的设备浏览器中处理与保存，不向服务器上传个人作答内容。
- 测试结果仅供自我了解与讨论参考，不构成任何专业或政治立场判定。
